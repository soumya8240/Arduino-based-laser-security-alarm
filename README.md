# Arduino-based-laser-security-alarm
I have used Arduino nano , a laser module, a buzzer , and a LDR sensor. I have connected the laser module with a power bank as a power supply and the Arduino and the other units are connected to the laptop, after i uploaded the code that I wrote in the Arduino IDE, The whole system is ready for action 
As when the lazer path (between the lazer module and ldr sensor ) is obstructed with anything the buzzer goes off.
I had setup the buzzer to play for a few seconds only but it can be changed to any amount of time or can be made to buzz continuously
